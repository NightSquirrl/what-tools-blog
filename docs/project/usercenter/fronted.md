## 项目的运行

### 依赖的安装
```shell
yarn
or
npm i
```
### 开发模式运行
```shell
yarn start
```

### 打包
```shell
yarn build
```

## 项目功能说明

>通过 session 实现自动登录功能

简单的实现登录和注册的功能

username:admin

password:12345678
